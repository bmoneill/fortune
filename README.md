# A simple fortune implementation

This is a simple fortune implementation that supports the same file
formats as the standard Linux/BSD versions.

## Installation

### Linux

```shell
make
sudo make install
```

## Usage

```shell
fortune [file] [separator]
```

Default separator is `'\n'`.
